| Error                         | Solution                                                                |
| ----------------------------- | ------------------------------------------------------------------------|
| `vendor/autoload.php` missing | Run `composer install`                                                  |
| `.env` missing                | Copy `.env.example` → `.env`                                            |
| `APP_KEY` missing             | Run `php artisan key:generate`                                          |
| SQLite database file missing  | Switch to MySQL in `.env` or create `database/database.sqlite`          |
| Sessions table missing        | Run `php artisan session:table` → `php artisan migrate`                 |
| `.test` domain not resolving  | Run Herd as Administrator, flush DNS: `ipconfig /flushdns`              |
| Internal Server Error         | Clear caches: `php artisan config:clear && php artisan cache:clear`     |
| git ownership doesn’t match   | git config --global --add safe.directory C:/Users/DELL/Herd/wmsu-paynet |
